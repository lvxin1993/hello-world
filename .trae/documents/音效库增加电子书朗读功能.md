# 音效库增加电子书朗读功能实现计划

## 1. 依赖安装
- 安装 `expo-speech` 库，用于实现文本转语音功能

## 2. 功能设计

### 2.1 电子书数据结构
- 添加电子书列表数据，包含标题、作者、内容、分类等信息
- 示例电子书数据：
  ```javascript
  const ebookList = [
    { id: '1', title: '静夜思', author: '李白', category: '诗歌', content: '床前明月光，疑是地上霜。举头望明月，低头思故乡。', thumbnail: '📖' },
    { id: '2', title: '春眠不觉晓', author: '孟浩然', category: '诗歌', content: '春眠不觉晓，处处闻啼鸟。夜来风雨声，花落知多少。', thumbnail: '📖' }
  ];
  ```

### 2.2 音色选择
- 支持多种朗读音色选择
- 示例音色数据：
  ```javascript
  const voices = [
    { id: 'zh-CN-XiaoxiaoNeural', name: '晓晓' },
    { id: 'zh-CN-YunxiNeural', name: '云希' },
    { id: 'zh-CN-YunyangNeural', name: '云阳' },
    { id: 'zh-CN-YunyeNeural', name: '云野' }
  ];
  ```

## 3. 代码修改

### 3.1 修改 SoundLibraryScreen.js
- 导入 `expo-speech` 库
- 添加电子书列表数据
- 添加音色选择数据
- 修改 UI 结构，添加电子书分类标签切换
- 添加电子书卡片组件
- 添加音色选择组件
- 实现 TTS 朗读逻辑

### 3.2 功能实现
- **分类切换**：支持在声音和电子书之间切换
- **电子书选择**：点击电子书卡片选择要朗读的内容
- **音色选择**：提供多种音色供用户选择
- **朗读控制**：支持播放、暂停、继续、停止功能
- **音量控制**：复用现有音量控制功能

## 4. UI 设计

### 4.1 分类标签
- 在音效库顶部添加分类标签切换（声音/电子书）
- 切换时显示对应的内容列表

### 4.2 电子书卡片
- 显示电子书标题、作者、分类
- 点击后进入朗读模式

### 4.3 朗读控制面板
- 显示当前朗读的电子书信息
- 音色选择下拉菜单
- 播放/暂停/停止按钮
- 音量控制滑块

## 5. 实现步骤

1. 安装 `expo-speech` 依赖
2. 修改 SoundLibraryScreen.js，添加分类切换逻辑
3. 添加电子书列表和音色数据
4. 实现电子书卡片组件
5. 实现 TTS 朗读功能
6. 添加音色选择功能
7. 测试功能完整性
8. 优化 UI 和用户体验

## 6. 预期效果

- 音效库支持切换到电子书分类
- 可选择不同的电子书进行朗读
- 可选择不同的音色进行朗读
- 支持播放控制和音量调节
- 整体 UI 风格与现有应用保持一致

# 为 DreamEntryCard 添加重力效果

## 概述
为 DreamEntryCard 组件添加交互式重力效果，当用户手指放到卡片上时，卡片会根据手指位置产生相应的倾斜和位移，模拟真实的重力感，增强用户体验。

## 实现方案

1. **添加触摸事件处理**
   - 使用 `PanResponder` 监听卡片上的触摸事件
   - 捕获手指在卡片上的位置坐标
   - 处理触摸开始、移动和结束事件

2. **创建动画值**
   - 添加新的 `Animated.Value` 用于卡片变换
   - 包括：旋转角度（rotateX, rotateY）、位移（translateX, translateY）
   - 初始值设置为0，代表无变换状态

3. **计算重力效果**
   - 根据触摸位置计算变换中心
   - 基于触摸点与卡片中心的距离计算变换强度
   - 使用平滑的数学函数（如平方根、归一化）确保效果自然

4. **应用变换效果**
   - 将计算出的变换应用到卡片的 `style.transform` 中
   - 结合现有的淡入和上滑动画
   - 确保变换原点设置正确

5. **优化用户体验**
   - 触摸结束时平滑恢复原状态
   - 限制变换范围，避免过度变形
   - 确保不影响现有点击事件的正常工作

## 代码修改点

1. **文件：src/components/DreamEntryCard.js**
   - 导入 `PanResponder` 组件
   - 初始化 `PanResponder` 并配置事件处理
   - 添加新的 `Animated.Value` 用于变换
   - 修改 `Animated.View` 的 `style`，添加重力变换效果
   - 确保 `PanResponder` 与现有 `TouchableOpacity` 事件兼容

## 技术细节

- 使用 `PanResponder` 替代直接的触摸事件，提供更完整的触摸状态管理
- 重力效果基于手指位置与卡片中心的相对距离计算
- 变换强度随距离中心的远近而变化，中心位置变换最强
- 使用 `Animated.spring` 或 `Animated.timing` 实现平滑过渡
- 确保所有变换都使用 `useNativeDriver: true` 以获得最佳性能

## 预期效果

- 当用户手指触摸卡片时，卡片会向触摸方向倾斜
- 触摸位置越靠近边缘，倾斜效果越明显
- 手指移动时，卡片会跟随调整倾斜方向
- 触摸结束时，卡片会平滑恢复到初始状态
- 整体效果自然流畅，增强卡片的立体感和交互性
